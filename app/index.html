First-Run Setup

Set up TalkieTxt once, then close the window and dictate from anywhere.

Choose your local Whisper model, verify the two macOS permission areas, and confirm the tray behavior. After that, the app behaves like a normal background utility.

1. Local model

Connect your local Whisper model

TalkieTxt checks its managed models folder automatically first. If it already finds a compatible model there, you do not need to browse manually.

2. macOS permissions

macOS will usually prompt when access is needed

Microphone access is requested the first time you record. Accessibility access is only needed for paste automation. You can still open the System Settings pages manually below if you want to verify or change them.

3. Background behavior

Close the window, keep the app running

The red close button hides the window. TalkieTxt keeps running in the tray so your hotkey continues working. Use the tray menu when you want to reopen settings or fully quit.

  • Hotkey: Option+Space
  • Primary mode: Hold to talk
  • Quit path: tray icon → Quit

Finish setup after your model status turns ready.
